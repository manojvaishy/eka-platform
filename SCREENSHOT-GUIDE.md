## 🎯 AB YE KARO (Screenshot Me Dekho)

### Step 1: "Deploy from a branch" Pe Click Karo

Screenshot me ye dikhega:
```
Source
[Deploy from a branch ▼]  ← YE DROPDOWN HAI, ISKO CLICK KARO!
```

**Kahan hai?**

- "Source" label ke neeche
- Blue button jisme "Deploy from a branch" likha hai
- Right side me down arrow (▼) hai

---

### Step 2: Dropdown Khulega

Jab click karoge, 2 options dikhenge:
```
┌─────────────────────────────┐
│ • Deploy from a branch      │ ← Currently selected (tick mark hoga)
│ • GitHub Actions            │ ← YE SELECT KARO!
└─────────────────────────────┘
```

**"GitHub Actions" pe click karo!**

---

### Step 3: Automatically Save Ho Jayega

Jab "GitHub Actions" select karoge:
- Dropdown band ho jayega
- "GitHub Actions" show hoga
- Neeche ek message aayega:
  ```
  "Use a workflow from your repository to build and deploy your site"
  ```
- Koi "Save" button nahi hai - automatic save hai!
