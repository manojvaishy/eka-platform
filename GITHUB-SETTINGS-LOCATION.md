
## 🔍 Agar Settings Button Nahi Dikh Raha

### Possible Reasons:

1. **Aap repository owner nahi ho**
   - Solution: Owner (manojvaishy) se access mango

2. **Repository private hai aur aapko access nahi**
   - Solution: Repository public karo ya access request karo

3. **Aap logged out ho**
   - Solution: GitHub me login karo

4. **Wrong repository khola hai**
   - Solution: Confirm karo URL: github.com/manojvaishy/EKA
