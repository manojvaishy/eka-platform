## ✅ Answer (Simple Hinglish):

### Part 1: Terminal Commands (Aapke Computer Pe)

**Ye commands run karo:**
```cmd
cd C:\Users\rupesh\EKA
git add .
git commit -m "Setup GitHub Pages deployment"
git push origin main
```

**Kya hoga?**
- Aapki files GitHub server pe upload ho jayengi
- Terminal me "done" type ka message aayega
- Koi error nahi aana chahiye

**Kitna time lagega?**
- 10-30 seconds

---

### Part 2: GitHub Website (Browser Me)

**Kahan jana hai?**

1. **Browser kholo** (Chrome, Edge, Firefox - koi bhi)

2. **Ye URL type karo:**
   ```
   https://github.com/manojvaishy/EKA/settings/pages
   ```
   (Ye direct aapko sahi page pe le jayega!)

3. **"Source" dropdown dhundo:**
   - Page ke beech me "Build and deployment" likha hoga
   - Uske neeche "Source" dropdown hoga
   - Currently "Deploy from a branch" selected hoga

4. **Dropdown click karo:**
   - 2 options dikhenge
   - "GitHub Actions" pe click karo

5. **Done!** Automatically save ho jayega

---

### Part 3: Wait Karo (2-3 Minutes)

**Kya ho raha hai background me?**
- GitHub automatically aapki website build kar raha hai
- HTML, CSS, JS files bana raha hai
- Server pe upload kar raha hai

**Kaise check karein?**

1. **Actions tab kholo:**
   ```
   https://github.com/manojvaishy/EKA/actions
   ```

2. **"Deploy to GitHub Pages" workflow dikhegi:**
   - 🟡 Yellow dot = Abhi chal raha hai (wait karo)
   - ✅ Green checkmark = Ho gaya! (success)
   - ❌ Red X = Error aaya (mujhe batao)

3. **Green checkmark aane ka wait karo** (2-3 minutes)

---

### Part 4: Website Check Karo

**Jab green checkmark aa jaye:**

1. **Ye URL kholo:**
   ```
   https://manojvaishy.github.io/EKA/
   ```

2. **Aapki website live hai! 🎉**
   - Landing page dikhega
   - "She Can Shine" heading
   - Purple-orange gradient
   - "Get Started" button

## 🆘 Agar Problem Aaye

### Problem 1: Terminal me error
**Error**: "git: command not found"
**Solution**: Git install karo: https://git-scm.com/download/win

### Problem 2: Settings button nahi dikh raha
**Solution**: Direct URL use karo:
```
https://github.com/manojvaishy/EKA/settings/pages
```

### Problem 3: Source dropdown me "GitHub Actions" nahi hai
**Reason**: Repository private hai
**Solution**: Repository public karo (Settings → General → Danger Zone → Change visibility)

### Problem 4: Website pe 404 error
**Solution**: 
- 5 minutes wait karo
- Browser refresh karo (Ctrl + F5)
- Incognito window me try karo

### Problem 5: Green checkmark nahi aa raha
**Solution**:
- Actions tab me workflow pe click karo
- Error logs padho
- Mujhe screenshot bhejo
